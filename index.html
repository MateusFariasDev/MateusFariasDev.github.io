Windows Forms, HTML, CSS e Javascript

Estou ainda no início do curso de Análise e Desenvolvimento de Sistemas, estamos utilizando C inicialmente para estudo de algoritmos e resolução de exercícios. Como esse começo de aprendizado de lingugagem fica aquela coisa presa em cmd, algo sem uma interface além de uma tela preta e textos. Foi buscando ir além da tela do prompt de comando que aprendi HTML, CSS e Javascript desde a faculdade de engenharia, fugindo das matérias de física e matématica, e estudando essas tecnologias.

Este foi um dos primeiros programas feito em C# em meu curso, agora no segundo semestre, além de C, estamos usando C# para a criação de softwares para Windows. Com essa necessidade de uma parte visual, eu foquei bastante em HTML, CSS, Javascript e principalmente Bootstrap, muito antes de conhecer C#.

Em um trabalho da matéria de Engenharia de Software I, criar um protótipo para um sistema de almoxarifado, utilizei tecnologias web para sua interface, demonstrando toda a elegância do Bootstrap da qual gosto muito.

Utilizando HTML, CSS e Javascript, o sistema poderia rodar em todas as plataformas através do navegador, seja em computadores, celulares ou tabletes, está inteiramente responsivel. Confira nas imagens o sistema rodando no computador e no celular.

Mas se a exigência fosse um sistema exclusivamente para computadores, poderia ser facilmente convertido para este propósito com Electronjs(estou sempre procurando, estudando por novas ferramentas e frameworks de python e javascript, são as duas linguagens que mais me fascinam).


Projetos com ESP32

No curso de Técnico em Mecatrônica, me interessei muito pela criação de projetos e protótipos utilizando Arduino. Programar no Arduino e CLP foi onde floresceu minha paixão por códigos e me fez seguir na área.

Conheci o ESP32 por meio de pesquisas na internet e conhecer toda sua capacidade, ele sendo uma versão muito melhor do Arduino, acabou me tornando um grande admirador do hardware.

Esses são alguns dos projetos, como este utilizando um display rodando Pong.

Controlar acionamento de luz via wi-fi

Através do navegador rodando tanto em celular ou computador é possível se comunicar com o ESP32, assim sendo possível fazer o acionamento da luz via wi-fi.

As fotos mostram o projeto, utilizando um relé conectado ao ESP32 para controlar o acionamento da luz e o hardware fazendo todo esse processo de conexão via html, e conforme a informção recebida, liga ou desliga a luz. O site pronto depois de dar toda a melhoria com Bootstrap, sendo o layout final do projeto.
